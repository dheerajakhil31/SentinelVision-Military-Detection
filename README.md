# SentinelVision: Military Object Detection System

## Overview

SentinelVision is a deep learning-based military surveillance system that detects and classifies objects such as tanks, aircraft, warships, soldiers, and weapons using YOLOv8.

## Features

* Real-time object detection on video streams
* Tactical dashboard with mission analytics
* Detection of 12 military classes
* GPU-accelerated inference

## Tech Stack

* Python
* YOLOv8 (Ultralytics)
* Flask (Backend)
* HTML, CSS (Frontend)

## Dataset

* Source: Kaggle Military Vehicles Dataset
* Custom labeled dataset with multiple military classes

## Results

* Achieved strong detection performance on real-world scenarios
* Accurate classification of vehicles, aircraft, and personnel

## How to Run

```bash
pip install -r requirements.txt
python app.py
```
